**What Happened:**

**User Impact:**

**Prevention:**
1. **Increase Concurrency Limit:** Raise the maximum concurrency setting for the `my-api` service to allow each instance to handle more requests concurrently. Monitor CPU and memory utilization to ensure instances aren't overloaded.
